# 고객 세그먼트 페르소나

## 목표

데이터 분석가 초보자를 위한 고객 세그먼트별 데이터 분석 접근 방법을 시각적으로 제시

## 프로세스

- 산업 선택
  - B2B, B2C
  - 도메인
- 고객 세그먼트 카테고리 선택
  - 연령대
  - 성별
  - 접속 주기
  - 결제 금액
- 고객 정보가 입력되면
  - 대략적인 페르소나 이미지 보여주기
  - 어떤 식으로 접근할지 접근법 선택
- 선택한 접근법에 따라 구체적인 방법 제시
  - 필요하다면 접근법에 필요한 추가 정보 입력
  - 선택 후 진행했을 때 예상되는 임팩트 보여주기
